### 📖 Project Purpose
**QurbaniHat** is a high-performance marketplace platform built to modernize the ethical sourcing of livestock. Utilizing the **Next.js 15 Turbopack** engine, it delivers a lightning-fast, visually rich experience that simplifies cattle selection for the holy occasion of Qurbani. The architecture focuses on secure user sessions via a custom proxy layer.

---

### 🛠 Tech Stack
* **Framework:** Next.js 15.0.3 (App Router)
* **Styling:** Tailwind CSS 3.4.1 
* **Authentication:** Better Auth (Google OAuth + Credentials)
* **Database:** MongoDB Atlas
* **Icons:** React-icons
* **Spinners:** React-Spinners (RingLoader)
* **Animations:** Lottie-react

### 🌟 Key Features
* **🛡️ Secure Proxy Architecture:** Implementation of `src/proxy.js` to handle real-time session validation and route protection.
* **📊 Smart Inventory:** Dynamic price-based sorting (Low-to-High / High-to-Low).
* **🎨 Premium UI:** Nature-inspired emerald palette with responsive layouts.

### 🚀 Setup and Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/fireflyurmi/qurbani-hat-project.git](https://github.com/fireflyurmi/qurbani-hat-project.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Setup:**
    Configure your `.env` with `MONGODB_URI` and `BETTER_AUTH_SECRET`.
4.  **Launch:**
    ```bash
    npm run dev
    ```
